# Sudoku Solver Using Java

## Overview
This project solves a 9x9 Sudoku puzzle using the Backtracking Algorithm.

## Features
- Solves Sudoku puzzles automatically
- Uses recursion and backtracking
- Validates rows, columns, and 3x3 grids

## Technologies
- Java
- Arrays
- Recursion
- Backtracking

## How to Run

```bash
javac Main.java
java Main
```
